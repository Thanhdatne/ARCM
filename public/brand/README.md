# Brand assets for Deposit UI

Place the real logo files here. The Deposit page reads these exact paths:

- `/brand/chains/ethereum-sepolia.svg`
- `/brand/chains/base-sepolia.svg`
- `/brand/chains/arbitrum-sepolia.svg`
- `/brand/chains/optimism-sepolia.svg`
- `/brand/chains/arc-testnet.svg`
- `/brand/tokens/usdc.svg`

You can use `.png` or `.webp` instead, but then update `components/deposit/DepositBridge.tsx` to match the file names.
